# Go-Api

## Build

```golang
go get

go build
```

And then run the `.exe` file

## Routes

`GET /` - Welcome page

`GET /todos` - Get all To-Dos

`POST /todos` - Post a new To-Do

`GET /todos/:todoId` - Get todo of :id
